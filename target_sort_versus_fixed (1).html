Classroom Setup Tips

  • Recommended: Install Touchmote (touchmote.net, free). Run as admin, pair Wiimotes (press 1+2). Set DolphinBar to Mode 4. Point at the bar — cursor follows with real Wii IR precision.
  • Solo play: Touchmote default "Mouse Cursor" mode gives continuous smooth aiming. B trigger = click.
  • 2-player versus: Switch Touchmote to "Touch Cursor" mode. Each Wiimote is an independent pointer. Each B trigger click aims AND shoots in one motion — click the item you want to hit. No continuous tracking between clicks, just point and click.
  • If aim feels reversed after sudden movement or aiming away from the bar, briefly point the remote at the floor then back at the screen — that resyncs the IR tracking.
  • Pull the drapes. Sunlight is infrared and confuses the Wii remote. Sunny windows are the #1 cause of shaky aim.
  • DolphinBar on TOP of the screen, centered, LEDs facing the kids. The DolphinBar IS the sensor bar.
  • Kids stand 2-3m back, facing straight at the screen. Batteries fresh, IR window on the remote unblocked.
  • Alternative for 3-4 remotes: DolphinBar Mode 3 (gamepad mode). Click REMOTES pill or press F9 to verify each remote is detected.
